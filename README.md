# Book Store Application Using .NET Core, C# React.js and MSSQL

## Solution Technologies

-  Asp.Net Core (.NET 7.0)
-  React 18
-  TypeScript
-  MS SQL SERVER
-  Entity Framework Core

### On Backend project

- CURD Operation by C#
-  Entities
-  Context
-  ORM
-  Http Methods
-  Swagger
